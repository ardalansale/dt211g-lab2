# Laboration 2 – ECMAscript

## Live Site
[Click here to view the site](https://dt211g-lab2.netlify.app/)

## Features
- Fetches course data from an API using JavaScript
- Displays courses in a sortable table
- Filters courses in real time using a search field

## Technologies
- Vite
- Node.js
- HTML, CSS, JavaScript
- Fetch API with async/await
